# Contributing

Thanks for helping improve this interview and FAQ repository.

## Good Contributions

- sharper answers to common interview questions
- new high-frequency beginner pitfalls
- better target-based review paths
- clearer wording for mock interview use

## Contribution Rules

- keep answers short enough for teaching, but complete enough for interviews
- tie theory back to project practice whenever possible
- avoid trivia-heavy questions that do not help learners grow
- update `CHANGELOG.md` for user-visible changes

## Local Checks

```bash
printf '\n' | python quiz.py --tag ai --limit 1
python study_plan.py --target automation --weeks 4
```
